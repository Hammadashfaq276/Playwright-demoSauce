This project is a UI automation testing framework developed for the Sauce Demo web application using Playwright.

The framework follows Page Object Model (POM) design to ensure clean, reusable, and maintainable test code and includes test execution reports.
This project is created for learning, practice, and SDET portfolio purposes.
